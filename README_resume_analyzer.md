# AI Resume Analyzer (Python)

A Python application that analyzes resumes and automatically extracts key candidate information such as skills, education, and experience details.

## How it works
1. The program takes a resume (text/PDF) as input.
2. It parses the unstructured text and identifies key sections — skills, education, experience.
3. Extracted information is organized and displayed in a structured, readable format.

## Skills Applied
- Python programming
- Text parsing and data extraction
- Basic NLP (Natural Language Processing) concepts
- Understanding of applied AI fundamentals
